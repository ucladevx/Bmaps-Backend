Events¶

Interacting with events collection in mlab

events.get_all_events()¶
/api/events:
Description:Returns JSON of all events in format that Mapbox likes
events.get_event_by_id(event_id)¶
Description:Returns JSON of singular event by event id
Route:/api/event-id/<event_id>
Parameters:event_id – value to match with event id’s to find specific event
events.get_event_by_name(event_name)¶
Description:Returns JSON of singular event by event name
Route:/api/event-name/<event_name>
Parameters:event_name – string to match with event names
events.get_event_categories()¶
Description:Returns JSON of all event categories used in all events.

Potential event categories: Crafts, Art, Causes, Comedy, Dance, Drinks, Film, Fitness, Food, Games, Gardening, Health, Home, Literature, Music, Other, Party, Religion, Shopping, Sports, Theater, Wellness Conference, Lecture, Neighborhood, Networking

Route:/api/event-categories
events.get_event_categories_by_date(date)¶
Description:Get cursor to all events on a certain day and get unique categories list for that day
Route:/api/event-categories-by-date/<date>
Parameters:date – can search by date in multiple formats (ex. 22 January 2018)
events.get_events_by_category(event_category)¶
Description:Returns JSON of currently existing event categories
Route:/api/event-category/<event_category>
Parameters:event_category – string to match with event categories
events.get_events_by_category_and_date()¶
Description:Returns JSON of events by event category starting on passed in date
Route:/api/events-by-category-and-date
events.get_events_by_date(date)¶
Description:Returns JSON of all events starting on passed in date
Route:/api/event-date/<date>
Parameters:date – can search by date in multiple formats (ex. 22 January 2018)
events.get_events_for_search(search_term, date)¶
Description:

Returns JSON of events on date that match search term in format that Mapbox likes
Route:

/api/search/<search_term>/<date>
Parameters:
  • search_term – a string to use to find events that contain that word
  • date – search in a certain date with raw date format or the following format -> 22 January 2018
events.get_events_today_for_search(search_term)¶
Description:Returns JSON of events today that match search term in format that Mapbox likes
Route:/api/search/<search_term>
Parameters:search_term – a string to use to find events that contain that word
