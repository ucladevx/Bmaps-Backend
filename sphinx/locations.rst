Locations
=========

Welcome to the Mappening Locations API! The simplest way to access the API is to go to the url `whatsmappening.io/api/v2/locations <https://whatsmappening.io/api/v2/locations>`_. See the explanation of the API below. The Locations API offers various ways to search and filter the location data and also allows for independent processing and filtering.
